Getting Started

Installation

npm install @nis2shield/vue-guard
# or
yarn add @nis2shield/vue-guard

Requirements

  • Vue 3.3+ or higher
  • @nis2shield/core (automatically installed)

Basic Setup

Use createNis2Plugin in your main entry file:

import { createApp } from 'vue';
import { createNis2Plugin } from '@nis2shield/vue-guard';
import App from './App.vue';

const app = createApp(App);

app.use(createNis2Plugin({
  auditEndpoint: '/api/nis2/telemetry/',
  idleTimeoutMinutes: 15,
  debug: import.meta.env.DEV
}));

app.mount('#app');

Using the useNis2 Composable

Use the composable to react to session state:

<script setup lang="ts">
import { watch } from 'vue';
import { useNis2 } from '@nis2shield/vue-guard';

const { isIdle, warningSeconds, logWarning } = useNis2();

watch(isIdle, (idle) => {
  if (idle) {
    window.location.href = '/logout?reason=idle';
  }
});
</script>

<template>
  <div v-if="warningSeconds" class="warning-banner">
    ⚠️ Session expires in {{ warningSeconds }} seconds
  </div>
</template>

Configuration Options

Option Type Default Description
auditEndpoint string required URL to send telemetry reports
idleTimeoutMinutes number 15 Minutes before idle timeout
debug boolean false Enable console logging
