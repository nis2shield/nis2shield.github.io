NIS2 Compliance for Express.js

Forensic logging, active defense, and security headers in a single app.use(). Production-ready NIS2 compliance middleware for Node.js applications.

npm install @nis2shield/express-middleware

Quick Start

import express from 'express';
import { nis2Shield } from '@nis2shield/express-middleware';

const app = express();

// One line to NIS2 compliance
app.use(nis2Shield());

app.listen(3000);

What's Included

v0.2.0 Features

Forensic Logging

Structured JSON logs with HMAC-SHA256 signing. NEW: File output with rotation.

Security Headers

HSTS, CSP, X-Frame-Options, Referrer-Policy, and Permissions-Policy out-of-the-box.

Rate Limiting

In-memory token bucket algorithm. Configurable windows, limits, and custom key generators.

GeoIP Blocking NEW

Block or allow traffic by country using MaxMind GeoLite2. Blocklist and allowlist modes.

Tor Detection NEW

Real Tor exit node detection using TorProject API with 6-hour caching.

PII Protection

AES-256 encryption for sensitive fields. IP anonymization for GDPR compliance.

Full Configuration

import { nis2Shield } from '@nis2shield/express-middleware';

app.use(nis2Shield({
  enabled: true,
  encryptionKey: process.env.NIS2_ENCRYPTION_KEY,
  integrityKey: process.env.NIS2_HMAC_KEY,
  
  logging: {
    enabled: true,
    anonymizeIP: true,
    encryptPII: true,
    piiFields: ['userId', 'email'],
  },
  
  activeDefense: {
    rateLimit: {
      enabled: true,
      windowMs: 60000,
      max: 100,
    },
    blockTor: true,
  },
  
  securityHeaders: {
    enabled: true,
    hsts: true,
    csp: "default-src 'self'",
  },
}));
