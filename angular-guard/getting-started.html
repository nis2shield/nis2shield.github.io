Getting Started

Installation

npm install @nis2shield/angular-guard
# or
yarn add @nis2shield/angular-guard

Requirements

  • Angular 16+ or higher
  • @nis2shield/core (automatically installed)

Basic Setup

Import Nis2Module in your AppModule:

import { Nis2Module } from '@nis2shield/angular-guard';

@NgModule({
  imports: [
    Nis2Module.forRoot({
      auditEndpoint: '/api/nis2/telemetry/',
      idleTimeoutMinutes: 15,
      debug: !environment.production
    })
  ]
})
export class AppModule {}

Using the Nis2Service

Inject the service to react to session state:

import { Component } from '@angular/core';
import { Nis2Service } from '@nis2shield/angular-guard';

@Component({
  selector: 'app-root',
  template: `
    <div *ngIf="warning$ | async as seconds">
      ⚠️ Session expires in {{ seconds }} seconds
    </div>
  `
})
export class AppComponent {
  warning$ = this.nis2.warning$;

  constructor(private nis2: Nis2Service) {
    this.nis2.isIdle$.subscribe(isIdle => {
      if (isIdle) {
        window.location.href = '/logout?reason=idle';
      }
    });
  }
}

Configuration Options

Option Type Default Description
auditEndpoint string required URL to send telemetry reports
idleTimeoutMinutes number 15 Minutes before idle timeout
debug boolean false Enable console logging
